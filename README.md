# Google-Maps-Access

This respository contains the data used to assess mobility in Seattle using the Google Maps Distance Matrix API. A breakdown of the files in the respository are summarized below.

* **Seattle_Transit_Access_09Dec.ipynb** - Contains the code to run the analysis and discussion of the findings. Additional information on data sources, etc. is contained here.

* **Assessing_Seattle_Mobility_06Dec.pdf** - Presentation slides from December 6th, 2018

* **Seattle_Mobility_Map_Visualization_09Dec.twb** - Tableau workbook with map visualizations, the workbook is also published on Tableau Public here: https://public.tableau.com/profile/daniel.white8128#!/vizhome/Seattle_Mobility_Map_Visualization_09Dec/RegionMap

* **Project_Proposal_20Nov.ipynb** - Project idea proposal

* **data/** - Contains all the data files needed to replicate the analysis.

* **json/** - Contains the raw JSON files from the API responses
